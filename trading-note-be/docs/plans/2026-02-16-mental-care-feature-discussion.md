## Round 1: 문제 정의 - 트레이더가 멘탈이 무너지는 순간

**Sujin Kang (행동심리학)**:
> 트레이딩에서 멘탈 붕괴가 일어나는 핵심 패턴은 3가지입니다:
>
> 1. **복수 매매 (Revenge Trading)** - 손실 후 즉시 무리한 진입. "이번엔 만회해야지"라는 심리. 통계적으로 복수 매매의 78%가 추가 손실로 이어집니다.
> 2. **FOMO (Fear of Missing Out)** - 급등하는 종목을 보고 계획 없이 뛰어듦. 특히 SNS에서 다른 트레이더의 수익 인증을 본 직후.
> 3. **틸트 (Tilt)** - 연속 손실 후 판단력이 흐려지는 상태. 포커에서 유래한 용어인데, 트레이딩에서도 동일하게 발생합니다. 3연패 이후 승률이 평소 대비 40% 하락한다는 연구가 있습니다.
>
> 핵심은 **감정이 최고조일 때 트레이더 자신은 그것을 인지하지 못한다**는 것입니다.

**Jason Hwang (프로덕트 전략)**:
> 기존 트레이딩 저널 앱들의 멘탈 케어를 분석해보면:
> - **TraderSync**: 감정 태그(행복/두려움/탐욕) 선택 → 단순 기록에 그침
> - **Edgewonk**: 감정 점수 1-10 → 통계에 반영하지만 실시간 개입 없음
> - **TradeZella**: 감정 필터링 → 과거 분석용, 예방 기능 없음
>
> 공통 문제: **사후 분석(Post-mortem)만 있고, 실시간 예방(Prevention)이 없다.**
> 모든 앱이 "거래 후 감정을 기록하세요"인데, 정작 필요한 건 **"지금 거래하면 안 됩니다"**입니다.

## Round 2: 차별화 방향 토론

### Approach A: "거래 전 멘탈 체크인" (Pre-Trade Mental Check-in)

**Sujin Kang**:
> 가장 임팩트 있는 접근은 **거래 진입 전 자기 점검**입니다. 인지행동치료(CBT)에서 사용하는 "생각 멈추기(Thought Stopping)" 기법을 차용할 수 있습니다.
>
> 새 거래를 기록하기 전에 3가지 질문:
> 1. "지금 감정 상태는?" (침착/흥분/불안/분노/FOMO)
> 2. "이 거래는 계획된 것인가?" (예/아니오)
> 3. "마지막 거래 결과가 이 판단에 영향을 주고 있나?" (예/아니오)
>
> 이 데이터가 쌓이면 "흥분 상태에서 진입한 거래의 평균 ROI: -3.2%"같은 인사이트를 줄 수 있습니다.

**Minjae Park (UX)**:
> 다만 이건 **마찰(Friction)**입니다. 30초 퀵 엔트리가 앱의 핵심 가치인데, 매번 3개 질문을 추가하면 UX가 무거워집니다. 선택적이어야 합니다.

**Jason Hwang**:
> 동의합니다. 단, **위험 상태가 감지될 때만** 자동으로 트리거되면 마찰이 아니라 보호 장치가 됩니다. 예를 들어 직전 거래가 큰 손실이었으면, 다음 거래 기록 시 자동으로 체크인이 뜨는 방식.

### Approach B: "나만의 트레이딩 원칙 대시보드" (Personal Trading Rules)

**Jason Hwang**:
> 모든 성공한 트레이더에게는 자기만의 원칙이 있습니다. 문제는 **원칙을 세워놓고도 감정적 순간에 잊어버린다**는 것입니다.
>
> 사용자가 자신의 트레이딩 원칙을 등록하고, 앱 곳곳에서 상기시켜주는 기능:
> - "손절은 -3%에서 무조건 실행한다"
> - "하루 3건 이상 거래하지 않는다"
> - "연속 2패 후에는 1시간 쉰다"
>
> 이 원칙들이 대시보드 상단에 항상 보이고, 원칙을 어길 수 있는 상황이 되면 경고합니다.

**Dongwoo Kim (엔지니어)**:
> 기술적으로 구현 가능합니다. 원칙을 룰 엔진처럼 구조화하면:
> - "하루 N건" → 오늘 거래 수 카운트해서 초과 시 경고
> - "연속 N패 후 휴식" → 최근 거래 결과 체크
> - "최대 손실 -N%" → 현재 진행중인 거래 ROI 체크
>
> 단순 텍스트 원칙(체크리스트)과 자동 감지 원칙(룰 기반)을 나누면 됩니다.

### Approach C: "감정 일기 + AI 패턴 분석" (Emotion Journal + Pattern Analysis)

**Hyejin Lee (데이터 사이언트)**:
> 매 거래마다 감정을 기록하면, 시간이 지나며 **감정-수익률 상관관계**를 분석할 수 있습니다.
>
> 예시 인사이트:
> - "당신은 '불안' 상태에서 거래할 때 평균 -2.1% 손실입니다"
> - "화요일 오전에 가장 침착하고, 금요일 오후에 가장 충동적입니다"
> - "3연승 후 과신 상태에서 진입한 거래의 75%가 손실입니다"
>
> 하지만 이건 **데이터가 충분히 쌓여야** 의미 있습니다. 최소 50건 이상의 거래가 필요합니다.

**Sujin Kang**:
> 패턴 분석은 강력하지만, 초기 사용자에게는 가치를 느끼기 어렵습니다. **즉시 가치를 느낄 수 있는 기능(A, B)**과 **장기적 인사이트(C)**를 조합해야 합니다.

## Round 3: 통합 전략 - "Trading Note만의 멘탈 케어 시스템"

**Jason Hwang**:
> 세 접근을 다 가져가되, 3단계로 나눕시다:
>
> **Phase 1 (즉시 가치)**: 트레이딩 원칙 + 거래 전 감정 체크
> **Phase 2 (데이터 축적)**: 감정-수익률 통계 대시보드
> **Phase 3 (AI 인사이트)**: 패턴 기반 실시간 경고

**Minjae Park**:
> Phase 1에서 핵심 UI를 정리하면:

### 1. 나의 트레이딩 원칙 (My Trading Rules)

```
위치: 대시보드 상단 또는 별도 /rules 페이지
+--------------------------------------------------+
| 나의 트레이딩 원칙                    [+ 원칙 추가] |
+--------------------------------------------------+
| [v] 손절은 -3%에서 무조건 실행한다                  |
| [v] 하루 3건 이상 거래하지 않는다       ← 자동 감지  |
| [v] FOMO로 진입하지 않는다                          |
| [ ] 연속 2패 후에는 1시간 쉰다         ← 자동 감지  |
+--------------------------------------------------+
| 오늘: 3/4 원칙 준수 (75%)                          |
+--------------------------------------------------+
```

> 원칙은 두 종류:
> - **체크리스트형**: 매 거래 시 수동으로 체크 ("이 거래는 계획된 것인가?")
> - **자동 감지형**: 시스템이 데이터로 판단 ("오늘 거래 3건 초과", "연속 2패 감지")

### 2. 거래 전 감정 체크인 (Pre-Trade Emotion Check)

```
위치: 새 거래 등록 모달 상단 (선택적, 위험 감지 시 강조)
+--------------------------------------------------+
| 지금 감정 상태는?                                   |
| [침착] [자신감] [흥분] [불안] [분노] [FOMO]          |
+--------------------------------------------------+
| (!) 직전 거래 -450,000원 손실. 복수 매매 주의.       |
|     잠시 쉬어가는 건 어떨까요?                       |
+--------------------------------------------------+
```

> **핵심 차별점**: 다른 앱은 거래 "후"에 감정을 기록하지만, 우리는 거래 "전"에 개입합니다. 그리고 시스템이 위험 상황을 감지해서 경고합니다.

### 3. 멘탈 케어 대시보드 (Mental Care Dashboard)

```
위치: /mental 또는 대시보드 내 탭
+--------------------------------------------------+
| 이번 주 멘탈 현황                                   |
|                                                    |
| 감정 분포: 침착 60% | 자신감 20% | 불안 15% | 분노 5%|
| 원칙 준수율: 82% (지난주 대비 +5%)                   |
| 위험 거래 비율: 12% (감정적 진입)                    |
+--------------------------------------------------+
| 감정별 수익률                                       |
| 침착: +2.3% avg | 자신감: +1.8% | 불안: -1.5%      |
| 흥분: -0.8%     | 분노: -4.2%   | FOMO: -3.1%      |
+--------------------------------------------------+
| 이번 주 인사이트                                    |
| "침착한 상태에서의 거래가 다른 감정 대비 3.5배        |
|  높은 수익률을 보이고 있습니다."                     |
+--------------------------------------------------+
```

## Round 4: 차별화 포인트 심화 토론

**Jason Hwang**:
> 경쟁 앱 대비 우리만의 차별점을 명확히 합시다.

**Sujin Kang**:
> 차별점 1: **사전 개입 시스템(Proactive Intervention)**
>
> 기존 앱: "오늘 감정은 어땠나요?" (사후 기록)
> Trading Note: "지금 거래하면 위험합니다" (사전 경고)
>
> 구체적으로:
> - 연속 손실 감지 → "쉬어가세요" 메시지 + 타이머
> - 비정상적 거래 빈도 → "오늘 평소보다 2배 많이 거래하고 있습니다"
> - 큰 손실 직후 즉시 진입 시도 → "복수 매매 패턴이 감지되었습니다"

**Hyejin Lee**:
> 차별점 2: **데이터 기반 자기 인식(Data-Driven Self-Awareness)**
>
> 감정을 기록만 하는 게 아니라, 감정과 수익률의 상관관계를 보여줍니다.
> "당신은 분노 상태에서 거래할 때 평균 -4.2% 손실" 같은 개인화된 통계는 다른 앱에 없습니다.
> 이것은 트레이더에게 가장 강력한 피드백입니다 - 본인의 데이터가 증명하니까요.

**Minjae Park**:
> 차별점 3: **비침투적 UI(Non-Intrusive Design)**
>
> 멘탈 케어 기능이 거래 기록의 속도를 해치면 안 됩니다.
> - 감정 선택은 이모티콘 원탭 (0.5초)
> - 원칙 체크는 기존 퀵 엔트리에 자연스럽게 녹임
> - 경고는 모달이 아닌 인라인 배너 (닫을 수 있음)
> - 강제가 아닌 넛지(Nudge) 방식
>
> "멘탈 케어가 있는 줄도 모르게 자연스럽게 보호받는 느낌"이 목표입니다.

**Dongwoo Kim**:
> 차별점 4: **트레이딩 원칙의 자동화(Automated Rule Enforcement)**
>
> 원칙을 텍스트로 적어두는 건 어디서나 할 수 있습니다. 하지만 원칙을 **시스템이 자동으로 체크**해주는 건 다릅니다.
> - "하루 3건" 원칙 → 3건째 거래 등록 시 자동 경고
> - "연속 2패 후 휴식" → 2연패 감지 시 타이머 제안
> - "최대 시드 비율 20%" → 투자금 입력 시 실시간 체크
>
> 트레이더가 원칙을 세우면, 앱이 원칙을 지켜줍니다.

**Jason Hwang**:
> 차별점 5: **거래 복기 워크플로우(Structured Review Flow)**
>
> 거래 상세 모달에서 "복기하기" 버튼 → 구조화된 복기 질문:
> 1. "이 거래의 진입 근거는 타당했는가?"
> 2. "감정이 판단에 영향을 주었는가?"
> 3. "같은 상황이 오면 다시 이 거래를 하겠는가?"
> 4. "배운 점은?"
>
> 이 복기 데이터가 쌓이면 → "같은 실수를 반복하는 패턴" 감지 가능.

## Round 5: 최종 차별화 전략 합의

**Jason Hwang**:
> 정리합니다. Trading Note 멘탈 케어의 핵심 슬로건:
>
> **"기록이 아닌 보호. 분석이 아닌 예방."**
>
> 경쟁사는 "감정을 기록하세요"이고, 우리는 "감정이 거래를 망치기 전에 막아줍니다"입니다.

### 최종 합의된 5대 차별화 포인트

| # | 차별점 | 경쟁사 | Trading Note |
|---|--------|--------|--------------|
| 1 | 사전 개입 | 사후 감정 기록 | 위험 감지 시 실시간 경고 |
| 2 | 데이터 증명 | 감정 태그만 | 감정별 수익률 통계로 자기 인식 |
| 3 | 비침투적 UX | 필수 입력 필드 | 원탭 감정 + 넛지 방식 경고 |
| 4 | 원칙 자동화 | 텍스트 메모 | 시스템이 원칙 준수 자동 체크 |
| 5 | 구조화 복기 | 자유 메모 | 질문 기반 복기 워크플로우 |

### Phase 1 구현 범위 (MVP)

1. **트레이딩 원칙 관리** - 원칙 CRUD + 체크리스트/자동감지 2종류
2. **거래 전 감정 체크인** - 6가지 감정 원탭 선택 (새 거래 등록 시)
3. **위험 상황 경고 배너** - 연속 손실, 비정상 빈도, 복수 매매 패턴 감지
4. **거래 후 복기 워크플로우** - 상세 모달에서 구조화 질문 기반 복기
5. **감정별 수익률 통계** - 멘탈 케어 대시보드 기본 차트

### Phase 2 (데이터 축적 후)

- 요일/시간대별 감정-수익률 히트맵
- "최적의 트레이딩 컨디션" 개인화 리포트
- 원칙 준수율 트렌드 차트

### Phase 3 (고급 기능)

- AI 기반 "오늘의 멘탈 상태 예측"
- 연속 패턴 감지 ("이 패턴은 3번째 반복입니다")
- 주간/월간 멘탈 리포트 자동 생성
